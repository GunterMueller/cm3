Known Problems with CM3 5.8 Release Candidates

RC2

  • Description:
    Many of the .M3SHIP files in the *-bin-ws-* archives contain absolute pathnames in some places where variable names are expected. Thus shipping to the correct user-defined position with the install.sh scripts will fail.
    Workaround:
    Before shipping each package, as the install.sh scripts does, call 'cm3 -build' to fix the adapt the shipping information to your actual setup.
    Fix:
    This will be fixed in RC3.
    Related Tickets:
    1052
  • Description:
    Missing CVSup on Solaris due to missing libz
    Workaround:
    Install libz.a and rebuild cvsup. Fix:
    Add libz to SYSTEM_LIBS and link it dynamically.
    Related Tickets:
    1048

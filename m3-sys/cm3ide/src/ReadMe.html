CM3-IDE: Critical Mass Modula-3 Integrated Development Environment

This source package (../src) builds the CM3-IDE web server application that together with your web browser serves as the IDE for CM3.
Documentation is available in the (../UsersGuide) folder, including the original word processing files used to create the PDF.
Original artwork for the logo et al is available in the (../src/rsrc/originalArtWork) folder.

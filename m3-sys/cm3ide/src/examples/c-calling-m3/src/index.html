Calling Modula-3 from C

This example demonstrates how to call Modula-3 procedures from C. C procedure that takes a single parameter which itself is a parameterless procedure that returns an integer. The C function call the passed the procedure, add one to the result and return the new value. The makefile denotes that the C code is in a file named Cstuff.c. The main module, CcallsM3, calls Cstuff.c via the interface Cstuff.

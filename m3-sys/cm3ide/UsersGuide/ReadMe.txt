ReadMe for CM3-IDE User's Guide
-------------------------------

The User's Guide was written using Microsoft Word 2003.  
Some of the figures were generated using Microsoft PowerPoint 2003.
The PDF file was generated from the Word document.

CM3_IDE_UsersGuide.doc = original Word Document
CM3_IDE_UsersGuide.pdf = pdf derived from the Word document

CM3_IDE_Screenshots_Figures.doc = screen (window) captures used as basis for the figures in the Word document

CM3_IDE_ComposedFigures.ppt = PowerPoint file containing some figures used in the Word document

---END---
